# Каталог товаров

Проект представляет собой систему управления категориями и товарами в интернет-магазине.

## Функционал

Реализованы следующие классы и возможности:

### Класс Product (Товар)
- Хранение информации о товаре:
  - Название (name)
  - Описание (description)
  - Цена (price)
  - Количество в наличии (quantity)

### Подкласс Товаров Smartphone (Смартфоны)
- Хранение информации о смартфонах:
  - Название (name)
  - Описание (description)
  - Цена (price)
  - Количество в наличии (quantity)
  - Производительность (efficiency)
  - Модель (model)
  - Объем встроенной памяти (memory)
  - Цвет (color)

### Подкласс Товаров LawnGrass (Газонная трава)
- Хранение информации о газонной траве:
  - Название (name)
  - Описание (description)
  - Цена (price)
  - Количество в наличии (quantity)
  - Страна-производитель (country)
  - Срок прорастания (germination_period)
  - Цвет (color)

### Класс Category (Категория)
- Хранение информации о категории товаров:
  - Название категории (name)
  - Описание категории (description)
  - Список товаров в категории (products)
- Подсчет общего количества категорий (category_count)
- Подсчет общего количества уникальных товаров (product_count)

## Пример использования

# Создание товаров
```
product1 = Product("Samsung Galaxy S23 Ultra", "256GB, Серый цвет, 200MP камера", 180000.0, 5)
product2 = Product("Iphone 15", "512GB, Gray space", 210000.0, 8)
smartphone2 = Smartphone("Iphone 15", "512GB, Gray space", 210000.0, 8, 98.2, "15", 512, "Gray space")
grass1 = LawnGrass("Газонная трава", "Элитная трава для газона", 500.0, 20, "Россия", "7 дней", "Зеленый")
```
# Создание категории
```
category = Category("Смартфоны", "Описание категории", [product1, product2])
```

# Получение информации
```
print(category.name)  # Название категории
print(len(category.products))  # Количество товаров
print(Category.category_count)  # Общее количество категорий
```
